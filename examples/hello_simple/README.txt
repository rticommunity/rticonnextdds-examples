============================================================
 Example Code -- Hello Simple
============================================================

Concept
-------
Example about how to write a simple application using RTI Connext DDS.
This is the example explained in the Getting Started manual. Please
refer to it for a more detailed explanation.

Example Description
-------------------
In this example we create two executable for a publisher (client) and a
subscriber (server). The publisher reads the standard input and publish it
using the Builtin Type String. On the other side, every sample received by
the subscriber is written into the standard output.
