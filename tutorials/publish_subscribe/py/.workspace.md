# Publish-Subscribe

This example provides an introduction to the publish-subscribe communication
model.

You will learn the following:

* Create a Topic and define its data type
* Create an application to publish data to your Topic
* Create an application to subscribe to your Topic and react to updates

Connext allows applications to share data by publishing and subscribing to
Topics. Data producers publish, or write, data. Data consumers subscribe to, or
read, data.

Connext automatically handles discovery and distribution, delivering
data from publishers directly to subscribers.

![Publisher-Subscriber Topic](https://community.rti.com/static/documentation/developers/_images/writer_reader_topic.png)

We will build a simple home automation system in Python in which:

* several sensors publish the status of a door or a window (whether it’s open or closed)
* an alert system subscribes to the status of all the sensors and prints a warning when a door or window is open.
